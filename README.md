# Online-salse-performance-analysis

This project analyzes customer behavior and sales performance using Python, pandas, matplotlib, and seaborn.

## Features
- Exploratory Data Analysis (EDA)
- Revenue and KPI analysis
- Customer retention analysis
- Category performance analysis
- Correlation and outlier analysis
- Data visualization with charts and heatmaps

## Technologies
- Python
- pandas
- matplotlib
- seaborn

## Skills Demonstrated
- Data analysis
- Customer analytics
- KPI reporting
- Trend analysis
- Correlation analysis
- Data visualization
